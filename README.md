

# HighFold 4

This repository is based on AlphaFold3. The related work is *HighFold4: extending Alpha*Fold3 to accurate cyclic peptide conformation prediction via custom chemical connectivity*

## Usage
### Install
Install dependent on alphafold3. [https://github.com/deepmind/alphafold](https://github.com/deepmind/alphafold)

### Copy files
Then copy files from this repository to Alphafold3 directory.


## Note
This repository is building, so it may have some bugs.
